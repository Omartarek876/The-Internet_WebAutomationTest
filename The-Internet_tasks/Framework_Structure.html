Selenium Automation Framework

Comprehensive documentation for the core utility classes.

Core Component

BaseDriver.java

This class is the foundation of the framework, managing the WebDriver instance and providing methods to initialize, retrieve, and quit the driver.

initializeDriver(String browser)

Initializes a new WebDriver instance (Chrome or Firefox).

Parameters:

  • browser (String): The browser to use, "chrome" or "firefox".

Return Value:

WebDriver

getDriver()

Returns the active WebDriver instance.

Return Value:

WebDriver

quitDriver()

Closes all browser windows and shuts down the WebDriver session.

Return Value:

void

Utility Classes

ElementUtils.java

Provides helper methods for common element interactions like clicking, typing, and selecting dropdown options, always using a `By` locator.

getElement(By locator)

Finds and returns a single visible `WebElement`.

Parameters: locator (By)

Return Value: WebElement

getElements(By locator)

Finds and returns a list of `WebElement` objects.

Parameters: locator (By)

Return Value: List<WebElement>

click(By locator)

Performs a click action on a located element.

Parameters: locator (By)

Return Value: void

clearAndSendKeys(By locator, String text)

Clears existing text and sends new text to an input field.

Parameters: locator (By), text (String)

Return Value: void

getText(By locator)

Gets the text content of a located element.

Parameters: locator (By)

Return Value: String

selectDropdownByText(By locator, String visibleText)

Selects an option from a dropdown by its visible text.

Parameters: locator (By), visibleText (String)

Return Value: void

selectDropdownByValue(By locator, String value)

Selects an option from a dropdown by its value attribute.

Parameters: locator (By), value (String)

Return Value: void

selectDropdownByIndex(By locator, int index)

Selects an option from a dropdown by its index.

Parameters: locator (By), index (int)

Return Value: void

checkCheckbox(By locator)

Checks a checkbox if it is not already selected.

Parameters: locator (By)

Return Value: void

uncheckCheckbox(By locator)

Unchecks a checkbox if it is currently selected.

Parameters: locator (By)

Return Value: void

selectRadioButton(By locator)

Selects a radio button if it is not already selected.

Parameters: locator (By)

Return Value: void

WaitUtils.java

Manages synchronization with explicit, implicit, and fluent waits to handle dynamic web pages.

waitForVisibility(By locator, int timeoutSeconds)

Waits for an element to be visible on the page.

Parameters: locator (By), timeoutSeconds (int)

Return Value: WebElement

waitForClickability(By locator, int timeoutSeconds)

Waits for an element to be clickable.

Parameters: locator (By), timeoutSeconds (int)

Return Value: WebElement

fluentWait(By locator, int timeoutSeconds, int pollingSeconds)

A flexible wait that polls for an element and can ignore exceptions.

Parameters: locator (By), timeoutSeconds (int), pollingSeconds (int)

Return Value: WebElement

waitForPresence(By locator, int timeoutSeconds)

Waits for an element to be present in the DOM.

Parameters: locator (By), timeoutSeconds (int)

Return Value: WebElement

waitForTextToBePresent(By locator, String text, int timeoutSeconds)

Waits for a specific text string to appear in an element.

Parameters: locator (By), text (String), timeoutSeconds (int)

Return Value: WebElement

waitForInvisibility(By locator, int timeoutSeconds)

Waits for an element to become invisible or to disappear from the DOM.

Parameters: locator (By), timeoutSeconds (int)

Return Value: boolean

setImplicitWait(int timeoutSeconds)

Sets a global implicit wait. Use with caution.

Parameters: timeoutSeconds (int)

Return Value: void
